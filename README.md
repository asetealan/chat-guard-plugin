# Chat Guard Plugin

Clean, shareable plugin artifact package with English documentation.

## Contents
- jar/ChatGuard-1.1.1.jar
- metadata/NOTES.md
- README.md
- SECURITY.md
- SANITIZATION_NOTES.md
- DISCLAIMER.md

## Platform
Paper/Spigot

## Purpose
Chat moderation layer for profanity, ad, and spam prevention.

## Install
1. Stop your server (or proxy for Velocity).
2. Copy the jar from jar/ into your plugins/ folder.
3. Start the service.
4. Adjust plugin config files generated by your own server environment.

## Scope
This repository intentionally excludes:
- docker-compose files
- database dumps
- private infrastructure files
- server-specific credentials

## Note
The jar binary is shipped as a tested runtime artifact. Deep localization or internal rebranding requires source-code rebuild.
